# Four Servo Motors Control

## Project Description
This project controls four servo motors using an Arduino Uno. The motors perform the Sweep movement for 2 seconds, then stop and hold at 90 degrees.

## Components
- Arduino Uno
- 4 Servo Motors

## Software
- Tinkercad
- Arduino (C++)

## Steps
1. Added four servo motors in Tinkercad.
2. Connected all servo motors to the Arduino Uno.
3. Connected the signal pins to digital pins D9, D10, D11, and D12.
4. Connected all power pins to 5V and all ground pins to GND.
5. Wrote the Arduino code using the Servo library.
6. Ran the Sweep motion for 2 seconds.
7. Made all servo motors stop at 90 degrees.
8. Tested the circuit using Tinkercad simulation.
